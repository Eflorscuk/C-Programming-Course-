Lista Encadeada / Lista Ligada / Linked Lista

A lista encadeada possui, além do seu valor, um ponteiro ligando para o próximo
elemento

-----    -----    -----
|1| | -> |2| | -> |3| | -> NULL
-----    -----    -----

* Serve para implementação de pilhas, filas e deques
